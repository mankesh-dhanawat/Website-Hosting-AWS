# Hosting-Website-using-AWS-server
### linux project 

create and launch the instance  
After establishing connection with the instance run the following commands  

`sudo apt update`  
`sudo apt install apache2`

`cd /var/www/html/`  
`sudo vim index.html`  
`sudo vim style.css`  
`sudo vim timer.js`  
`sudo vim utilis.css`
